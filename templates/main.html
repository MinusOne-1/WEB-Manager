О Проекте

Что это и зачем?

Данный сайт - учебный проект по темам WEB`а в Яндекс.Лицее, представляющий из себя сайт, моделирующий управление поставками в компании по производству дронов. Основан на наработках, оставшихся с финала Олимпиады НТИ по профилю "Автоматизация бизнес-процессов".

Навигация в главном меню

Главное меню, расположеное сверху, имеет несколько разделов, каждый из ниж ведёт к тому или иному месту на сайте. Чаще всего это формы, где можно посмотреть информацию из базы данных, либо занести новую информацию в базы данных.

  1. Главная

    2. Ведёт на главную страницу с описанием сайта. То есть сюда.

  2. Регистрация поступления

    3. Ведёт на страницу "Регистрация поступления". Там можно добавить в базу данных "поступление комплектующи", которые приходят на наш воображаемый склад. Для этого в форме нужно будет указать Номер поступления, дату, а так же вид поступивших комплектующих.

  3. Заявка на поступление

    4. Выпадающее меню, имеет два пункта: "Новая" и "Посмотреть все заявки". Первый пункт позволяет занести в базу данных новую воображаемую заявку на сборку дрона. Второй пункт позволяет посмотреть все заявки, которые находятся в базе данных на данный момент.

  4. Остатки

    5. Ведёт на страницу, где можно посмотреть остатки на всех воображаемых складах.

  5. Склады

    6. Ведёт на страницу, где можно посмотреть место расположение несуществуеющих складов.

Контакты

Внизу страницы находится ссылка на профиль автора проекта на ГитХабе.
